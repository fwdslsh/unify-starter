Data Import Components

Modern CSS Features

This site showcases cutting-edge CSS:

  • CSS Custom Properties for theming
  • View Transitions API for smooth navigation
  • Container Queries for responsive components
  • CSS Grid and Flexbox for layouts
  • Modern color functions (oklch, color-mix)

Layout Discovery

Pages automatically use appropriate layouts:

  • .layouts/default.html - Default layout
  • .layouts/blog.html - Blog posts
  • .layouts/products.html - Product pages

Performance Features

Built for modern web standards:

  • Incremental builds and dependency tracking
  • Asset optimization and minification
  • Modern viewport units (dvh, cqi)
  • Prefers-reduced-motion support

🚀 Try These Commands

Experience Unify's development workflow and build system:

🔧 Development Mode

Start the development server with live reload and instant updates:

# Start development server
npm run dev

# With custom port
npx @fwdslsh/unify serve --port 8080

# Watch for changes
npx @fwdslsh/unify watch

💡 Try editing any file and watch the browser auto-refresh!

🚀 Production Build

Build optimized static files ready for deployment:

# Basic build
npm run build

# With pretty URLs (/about.html → /about/)
npm run build:pretty

# Full production build
npm run build:production

📦 Output goes to dist/ folder
