About Unify Demo

Welcome to the comprehensive demonstration of Unify, the modern static site generator that combines simplicity with powerful features.

🎯 Our Mission

To provide developers with a zero-configuration static site generator that doesn't sacrifice functionality for simplicity.

⚡ Performance First

Built with Bun-native APIs for lightning-fast builds and development experience.

🧩 Component-Based

Create reusable components with slots, templates, and scoped styles/scripts.

📝 Content-Focused

Support for Markdown with frontmatter, automatic layout discovery, and pretty URLs.

What This Demo Showcases

This starter project demonstrates every feature of Unify according to the latest specification:

  • Layout System: Nested layouts, fallback layouts, and layout discovery
  • DOM Templating: Modern data-import, data-target, and <slot> elements
  • Markdown Processing: YAML frontmatter with layout overrides
  • Component Architecture: Reusable components with isolated styles and scripts
  • Asset Management: Automatic asset discovery and copying
  • Development Tools: Live reload server with incremental builds
  • Build Options: Pretty URLs, minification, and custom asset copying

🚀 Getting Started

To start building with this demo:

  1. Clone this repository
  2. Run npm install to install dependencies
  3. Run npm run dev to start the development server
  4. Edit files in src/ and see changes instantly
  5. Run npm run build to build for production
