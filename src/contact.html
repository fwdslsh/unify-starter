Frequently Asked Questions

Is Unify free to use?

Yes! Unify is completely free and open source under the MIT license.

What's the difference between Unify and other static site generators?

Unify focuses on simplicity and uses familiar HTML templating patterns. No complex configuration required!

Can I use Unify for commercial projects?

Absolutely! Unify is MIT licensed, so you can use it for any project, including commercial ones.

How do I get started with Unify?

Clone this starter repository, run npm install, then npm run dev to get started!
